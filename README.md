# Password Generator

A sleek, responsive Password Generator web application built with React and Tailwind CSS that generates secure, random passwords. It features an interactive slider to adjust password length and checkboxes to dynamically include numbers and special characters based on user preferences.

## 🚀 Features

- **Dynamic Customization:** Instantly adjust password length (from 8 to 100 characters) using an interactive slider.
- **Granular Security Controls:** Checkboxes to include or exclude numerical digits (`0-9`) and special characters (`@+_-*&#$`).
- **Performance Optimized:** Built utilizing React's `useCallback` and `useEffect` hooks for efficient state management and automated password regeneration.
- **Modern UI/UX:** Styled with Tailwind CSS v4 featuring a dark-themed, clean user interface.

## 🛠️ Tech Stack

- **Frontend Framework:** React (Vite)
- **Styling:** Tailwind CSS v4
- **Language:** JavaScript (ES6+)
